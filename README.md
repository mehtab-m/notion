# My Notion Dashboard

A full-stack personal dashboard app built with React (Vite) + Node.js (Express) + MongoDB + Mongoose + Multer.

## Prerequisites

- Node.js 18+
- MongoDB running locally on port 27017

## Setup & Run

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend (in a new terminal)

```bash
cd frontend
npm install
npm run dev
```

## Open

http://localhost:5173
